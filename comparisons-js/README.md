# Javascript Comparison Operators

| Operator      |   Meaning                  |
| ------------- |:--------------------------:|
| ==            | equality                   |
| ===           | strict-equality            |
| !=            | not equal to               |
| <=            | less than or equal to      |
| >=            | greater than or equal to   |
| <             | less than                  |
| >             | greater than               |


&& is use to symbolize AND

|| is used to symbolize OR
